# Commands Checking

With this plugin you will be able
to create commands that will work in a certain place and only.

## Features of Commands Checking

- Hex color(1.16+)
- Placeholder API Support
- 8 Actions
- World selection for the commands
- Radius selection
- actions / deny actions
